# [Leggjasaman](https://open.kattis.com/problems/leggjasaman)

This word problem boils down to adding two numbers. The edge cases do not require any special consideration.

## Psuedo Code
```
INPUT arnar, hannes

PRINT arnar + hannes
```

## Time Complexity
This is exactly $O(1)$ = $\Omega$(1) = $\Theta(1)$.
